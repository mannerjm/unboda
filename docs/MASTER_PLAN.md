# UNBODA MASTER PLAN v1.0

## 1. 프로젝트 비전

운보다는 사주만 제공하는 서비스가 아니라,
사주·타로·꿈해몽 등 다양한 운세 콘텐츠를 하나로 연결하는
AI 기반 통합 운세 플랫폼을 목표로 한다.

핵심 목표:

- 사용자가 이해하기 쉬운 해석
- 계산과 해석의 분리
- 불안을 조장하지 않는 상담
- 실제 생활에 도움이 되는 조언
- 신뢰할 수 있는 명리 계산
- 모바일 중심의 편리한 사용자 경험

---

## 2. 제품 개발 원칙

### Engine First

모든 계산은 `app/lib` 내부의 엔진에서 처리한다.

UI에서는 계산하지 않는다.

AI는 계산하지 않고,
엔진에서 계산된 결과를 받아 설명만 한다.

### 기능별 파일 분리

```text
app/lib
├─ manse.ts
├─ nobles.ts
├─ elements.ts
├─ weights.ts
├─ strength.ts
├─ yongsin.ts
├─ daewoon.ts
├─ sewoon.ts
├─ tarot.ts
├─ dream.ts
└─ compatibility.ts


- 사주 원국 계산 엔진
- 오행 분포 계산
- 신강·신약 판정
- 용신 분석
- 격국 분석
- 용신 회귀 테스트
- 격국 회귀 테스트
- 식상 특수 출력 케이스 검증
- 인성 특수 출력 케이스 검증
- TypeScript 전체 타입 검사 통과

### 현재 검증 결과

- 격국 회귀 테스트 이상 후보: 0건
- 용신 경계 사례: 13건 검토 완료
- 식상 특수 출력 케이스: 216건
- 인성 특수 출력 케이스: 49건

현재 원국 기반 핵심 분석 엔진은 1차 안정화 단계까지 완료했다.

## 4. 앞으로의 개발 순서

### Phase 1. 대운 엔진

`app/lib/daeun.ts`

구현 목표:

- 대운 순행·역행 판정
- 대운 시작 나이 계산
- 10년 단위 대운 간지 생성
- 원국과 대운의 오행 관계 계산
- 대운별 용신·기신 영향 분석

---

### Phase 2. 세운 엔진

`app/lib/sewoon.ts`

구현 목표:

- 연도별 세운 간지 계산
- 원국·대운·세운 관계 분석
- 합·충·형·파·해 변화 반영
- 오행 증감 분석
- 용신 활성도 분석

---

### Phase 3. 운세 통합 분석

원국 + 대운 + 세운을 하나의 분석 구조로 통합한다.

구현 목표:

- 현재 대운 판단
- 현재 세운 판단
- 기회 요인 분석
- 주의 요인 분석
- 시기별 변화 설명
- 직업·재물·연애·건강 등 주제별 해석 기반 생성

---

### Phase 4. AI 해석 고도화

계산 엔진은 사실 데이터를 생성하고,
AI는 계산 결과만 받아 자연어로 설명한다.

구현 목표:

- 원국 해석 프롬프트 개선
- 대운 해석 프롬프트 추가
- 세운 해석 프롬프트 추가
- 원국·대운·세운 종합 해석
- 서로 충돌하는 해석 방지
- 사용자에게 이해하기 쉬운 상담형 설명 제공

---

### Phase 5. 서비스 확장

사주 핵심 엔진 안정화 이후 순차적으로 확장한다.

1. 궁합
2. 타로
3. 꿈해몽
4. 기타 운세 콘텐츠
