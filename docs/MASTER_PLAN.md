# UNBODA MASTER PLAN v1.0

## 1. 프로젝트 비전

운보다는 사주만 제공하는 서비스가 아니라,
사주·타로·꿈해몽 등 다양한 운세 콘텐츠를 하나로 연결하는
AI 기반 통합 운세 플랫폼을 목표로 한다.

핵심 목표:

- 사용자가 이해하기 쉬운 해석
- 계산과 해석의 분리
- 불안을 조장하지 않는 상담
- 실제 생활에 도움이 되는 조언
- 신뢰할 수 있는 명리 계산
- 모바일 중심의 편리한 사용자 경험

---

## 2. 제품 개발 원칙

### Engine First

모든 계산은 `app/lib` 내부의 엔진에서 처리한다.

UI에서는 계산하지 않는다.

AI는 계산하지 않고,
엔진에서 계산된 결과를 받아 설명만 한다.

### 기능별 파일 분리

```text
app/lib
├─ manse.ts
├─ nobles.ts
├─ elements.ts
├─ weights.ts
├─ strength.ts
├─ yongsin.ts
├─ daewoon.ts
├─ sewoon.ts
├─ tarot.ts
├─ dream.ts
└─ compatibility.ts


- 사주 원국 계산 엔진
- 오행 분포 계산
- 신강·신약 판정
- 용신 분석
- 격국 분석
- 용신 회귀 테스트
- 격국 회귀 테스트
- 식상 특수 출력 케이스 검증
- 인성 특수 출력 케이스 검증
- TypeScript 전체 타입 검사 통과

### 현재 검증 결과

- 격국 회귀 테스트 이상 후보: 0건
- 용신 경계 사례: 13건 검토 완료
- 식상 특수 출력 케이스: 216건
- 인성 특수 출력 케이스: 49건

현재 원국 기반 핵심 분석 엔진은 1차 안정화 단계까지 완료했다.

## 4. 앞으로의 개발 순서

### Phase 1. 대운 엔진

`app/lib/daeun.ts`

구현 목표:

- 대운 순행·역행 판정
- 대운 시작 나이 계산
- 10년 단위 대운 간지 생성
- 원국과 대운의 오행 관계 계산
- 대운별 용신·기신 영향 분석

---

### Phase 2. 세운 엔진

`app/lib/sewoon.ts`

구현 목표:

- 연도별 세운 간지 계산
- 원국·대운·세운 관계 분석
- 합·충·형·파·해 변화 반영
- 오행 증감 분석
- 용신 활성도 분석

---

### Phase 3. 운세 통합 분석

원국 + 대운 + 세운을 하나의 분석 구조로 통합한다.

구현 목표:

- 현재 대운 판단
- 현재 세운 판단
- 기회 요인 분석
- 주의 요인 분석
- 시기별 변화 설명
- 직업·재물·연애·건강 등 주제별 해석 기반 생성

---

### Phase 4. AI 해석 고도화

계산 엔진은 사실 데이터를 생성하고,
AI는 계산 결과만 받아 자연어로 설명한다.

구현 목표:

- 원국 해석 프롬프트 개선
- 대운 해석 프롬프트 추가
- 세운 해석 프롬프트 추가
- 원국·대운·세운 종합 해석
- 서로 충돌하는 해석 방지
- 사용자에게 이해하기 쉬운 상담형 설명 제공

---

# Phase 5. 무료 / 유료 분석 구조

목표:
무료 사용자가 운보다의 정확성과 가치를 충분히 경험한 뒤
자연스럽게 유료 심층 분석으로 이동하도록 만든다.

## 무료 제공
- 기본 사주 원국
- 오행 분포
- 신강 / 신약
- 용신 요약
- 현재 대운 요약
- 현재 세운 요약
- 현재 가장 중요한 운의 핵심 요약

## 유료 제공
- 재물운 심층 분석
- 직업 / 사업운 심층 분석
- 연애 / 관계 분석
- 올해 운세 상세 분석
- 10년 대운 분석
- 종합 인생 리포트

원칙:
무료 분석의 품질을 의도적으로 낮추지 않는다.

무료에서는:
"현재 어떤 흐름인가"

유료에서는:
"왜 그런가"
"언제 강해지는가"
"어떤 기회와 위험이 있는가"
"어떻게 대응하면 좋은가"

까지 제공한다.

---

# Phase 6. 개인 맞춤 운세 추천 엔진

목표:
모든 사용자에게 같은 상품을 보여주는 것이 아니라
현재 사주와 운의 흐름에 따라 필요한 분석을 자동 추천한다.

## 구현 항목
- 현재 가장 중요한 운 판정
- 직업 변화 신호
- 재물 변화 신호
- 관계 변화 신호
- 건강 / 소모 신호
- 대운 전환기 신호
- 강한 세운 변화 신호
- 용신 활성 시기
- 합·충·형·파·해 주요 변화 신호

## 추천 예시

직업 변화 신호 강함
→ 직업 / 사업운 분석 추천

재물 변화 신호 강함
→ 재물운 심층 분석 추천

관계 변화 신호 강함
→ 연애 / 관계 분석 추천

대운 전환기 접근
→ 10년 대운 분석 추천

목표:
운보다가 상품을 판매하기 전에
사용자에게 현재 가장 중요한 문제를 먼저 발견해준다.

---

# Phase 7. 유료 심층 리포트

목표:
사용자의 실제 고민을 해결하는 단품 유료 상품을 만든다.

## 핵심 상품
- 재물운 심층 분석
- 직업 / 사업운
- 연애 / 관계 분석
- 올해 운세
- 10년 대운 분석
- 종합 인생 리포트

각 리포트는 다음 구조를 기본으로 한다.

1. 현재 상황
2. 명리학적 근거
3. 대운 흐름
4. 세운 흐름
5. 중요한 시기
6. 기회 요인
7. 주의 요인
8. 현실적인 행동 가이드

---

# Phase 8. AI 명리 질문 상담

목표:
리포트를 읽은 사용자의 추가 고민을
개인 맞춤 AI 명리 상담으로 연결한다.

## 질문 예시

- 사업을 시작해도 될까요?
- 지금 투자해도 될까요?
- 이 사람과 결혼해도 될까요?
- 언제 움직이는 것이 좋을까요?

## AI 상담 기본 데이터
- 원국
- 신강 / 신약
- 용신
- 대운
- 세운
- 합·충·형·파·해
- 오행 변화
- 용신 활성도

## 수익 모델
- 질문 단품
- 질문 묶음 상품
- 구독 회원 질문 제공

- 세운 변화 감지

- 용신 활성 시기 감지
- 강한 합·충·형·파·해 변화 감지

## 사용자 흐름

운 변화 발생
→ 알림
→ 무료 핵심 요약
→ 상세 분석
→ 필요한 경우 유료 심층 분석

# Phase 16. 통합 운명 AI 상담 엔진

목표:
사용자가 서비스를 직접 선택하지 않아도
자연어 고민만 입력하면 필요한 분석을 자동 연결한다.

## 예시

사용자:
"올해 카페를 열어도 될까요?"

운보다:
1. 질문 의도 분석
2. 사주 사업운 분석
3. 현재 대운 분석
4. 현재 / 향후 세운 분석
5. 필요하면 개업 택일 추천
6. 사업 파트너가 있다면 궁합 분석
7. 매장이 있다면 풍수 분석 연결

## 통합 가능한 전문 엔진
- 사주
- 대운
- 세운
- 궁합
- 택일
- 작명 / 개명
- 이사
- 개업 / 창업
- 타로
- 풍수

각 엔진의 계산 결과와 AI의 설명을 분리하고
계산 결과를 우선 근거로 사용한다.
